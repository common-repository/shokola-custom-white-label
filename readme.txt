Set WordPress as "White Label" and personalize the login page. See <code>Settings / Custom Shokola</code> to configure.

== Description ==

Customize the login page with following settings :

* Image logo
* Link on the logo
* Background image
* Background position, repetition and cover parameter
* Favicon image
* Classic or Material style
* Light or Dark ambience
* Buttons and borders color
* Logo background color

Also :

* Removes WordPress logo and version in the back-office
* Removes WordPress welcome widget and WordPress News Widget on the back-office dashboard
* Automatically check 'Remember me' on the login page
* Removes 'Proudly powered by WordPress' on the login page (mouse hovering on the logo)

== Installation ==

= From your WordPress dashboard =

1. Go to 'Plugins > Add New'
2. Search for `Shokola Custom and WhiteLabel`
3. Activate Shokola Custom and WhiteLabel from your Plugins page.
4. Configure the plugin by going to the `Shokola Custom` menu that appears in your settings menu

= From WordPress.org =

1. Download Shokola Custom and WhiteLabel.
2. Upload the `shokola-custom-whitelabel` directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate Shokola Custom and WhiteLabel from your Plugins page.
4. Configure the plugin by going to the `Shokola Custom` menu that appears in your settings menu
